# Unbooked Media Assets

Public repository for media assets used across Unbooked projects.

## Contents

- `videos/` - Video assets for web applications
- `images/` - Images, logos, and graphics
- `audio/` - Audio files and sound effects

## Usage

Assets are served via jsDelivr CDN for fast, reliable delivery:

```
https://cdn.jsdelivr.net/gh/Unbookd/unbooked-media@main/path/to/file
```

## Structure

```
unbooked-media/
├── videos/
│   └── score-card-model.mp4
├── images/
│   └── logos/
└── audio/
```
